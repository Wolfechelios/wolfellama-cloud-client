# Repo Runner Launch Copy

## Product Headline

Run GitHub Apps Without Fighting the Terminal

## Subheadline

WolfeLlama helps you paste a GitHub repo, organize it as a project, detect its setup, manage environment fields, read plain-English logs, and ask AI for help when something breaks.

## Short Pitch

Most people find useful GitHub projects but get blocked by setup instructions. WolfeLlama turns those repos into guided projects with simple controls, saved project memory, and AI help built in.

## Best One-Line Pitch

Paste a GitHub repo. WolfeLlama helps you run it.

## Best Longer Pitch

WolfeLlama is a no-code GitHub repo runner for people who want to use open-source apps without becoming terminal experts. It organizes repos into a project library, detects app setup, helps manage env fields, keeps readable logs, and connects errors to AI explanations.

## Who It Is For

- Non-coders who want to use GitHub apps.
- AI builders using tools from ChatGPT, Bolt, Cursor, Lovable, and GitHub.
- Indie hackers testing open-source projects.
- Students learning app setup.
- Local AI users trying to run model tools.
- Small business owners who want open-source utilities without setup confusion.

## Main Pain

GitHub projects usually assume the user understands:

- package managers
- dependencies
- setup commands
- launch commands
- ports
- environment files
- terminal logs

WolfeLlama turns that into a guided workflow.

## Feature Bullets

- Paste a GitHub URL.
- Save repos as projects.
- Detect project type.
- Detect package manager.
- Show simple project controls.
- Manage environment fields.
- Keep project logs.
- Translate common errors into plain English.
- Send logs to local or cloud AI for help.
- Keep projects organized in one library.
