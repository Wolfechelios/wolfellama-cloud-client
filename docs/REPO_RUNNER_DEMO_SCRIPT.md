# Repo Runner Demo Script

Use this to record a short demo video or test the product flow.

## Demo Title

Run GitHub Apps Without Fighting the Terminal

## Opening Line

Found a cool GitHub project but do not know how to run it? WolfeLlama gives you a simple Repo Runner workspace that guides you through the process.

## Scene 1: Open WolfeLlama

Show the sidebar.

Point out:

- Repo Runner is the first workspace.
- Command Room is available for AI help.
- Agent Mode is available for planning.
- Hardware tools are optional.

Narration:

WolfeLlama is built around one simple idea: paste a GitHub repo and get help turning it into a working app.

## Scene 2: Add a Repo

Paste a GitHub URL into Repo Runner.

Click Add Repo.

Narration:

Instead of copying terminal setup instructions by hand, WolfeLlama saves the repo as a project and starts organizing what it needs.

## Scene 3: Project Library

Show the saved project in the library.

Narration:

Every repo becomes a saved project with its own setup, logs, environment fields, and status.

## Scene 4: Detect Project Type

Click Detect.

Show the facts grid.

Narration:

WolfeLlama identifies the kind of app, the package manager, the setup command, and the best launch option.

## Scene 5: Environment Fields

Add sample fields in the Environment Fields area.

Narration:

If the project needs keys or settings, WolfeLlama keeps those values organized per project.

## Scene 6: Project Controls

Show Install, Run, Stop, Restart, and Explain Logs With AI.

Narration:

The goal is simple controls instead of making users memorize commands.

## Scene 7: Logs in Plain English

Show the log card.

Narration:

When something breaks, WolfeLlama keeps the raw log but also explains what it means in plain English.

## Scene 8: AI Help

Click Explain Logs With AI.

Show Command Room with the log explanation prompt.

Narration:

WolfeLlama can pass logs to your local or cloud AI model so you can understand the next fix.

## Closing Line

WolfeLlama turns GitHub repos into guided projects, so builders and beginners can run open-source apps without getting stuck in setup confusion.

## Best Video Length

60 to 90 seconds.

## Strongest Selling Line

Paste a GitHub repo. WolfeLlama helps you run it.
